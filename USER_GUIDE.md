# BiliBili Sync for Obsidian 使用指南

把 B 站收藏夹同步到 Obsidian Vault，可选拉取视频逐字稿（字幕），并用 AI 自动生成总结写在笔记顶部。

---

## 一、下载安装

### 1. 下载插件

下载插件安装包：[bili2obsidian.zip](https://static.2notion.com/bili2notion/bili2obsidian.zip)

<!-- TODO: 补充下载页面截图 -->

### 2. 解压并安装

1. 解压 `bili2obsidian.zip`，得到 `bili2obsidian` 文件夹
2. 打开你的 Obsidian Vault 所在目录，找到 `.obsidian/plugins/` 文件夹
   - macOS：`你的Vault目录/.obsidian/plugins/`
   - Windows：`你的Vault目录\.obsidian\plugins\`
   - 如果 `plugins` 文件夹不存在，手动创建一个
3. 把 `bili2obsidian` 文件夹整个复制到 `plugins/` 目录下

最终结构：

```
你的Vault/
└── .obsidian/
    └── plugins/
        └── bili2obsidian/
            ├── main.js
            ├── manifest.json
            └── styles.css
```

> 提示：`.obsidian` 是隐藏文件夹。macOS 按 `Cmd + Shift + .` 显示隐藏文件，Windows 在文件资源管理器中勾选「显示隐藏的项目」。

### 3. 启用插件

1. 打开 Obsidian
2. 进入 **设置 → 第三方插件**
3. 如果提示「安全模式」，点击 **关闭安全模式**
4. 点击 **刷新** 按钮，找到 **BiliBili Sync**
5. 打开开关启用插件

## 二、配置插件

启用后，进入 **设置 → BiliBili Sync** 进行配置。

### 1. 授权码

- 免费试用 100 篇视频。
- 用完后会出现「授权码」输入框：填入授权码 → 点击「验证」按钮 → 显示「✅ 授权码验证成功！」。
- 第一次验证会把授权码绑定到当前电脑（防止多人共享同一个授权码）；卸载重装、切换 B 站账号都不影响绑定，只有换电脑需要联系作者重新解绑。
- 没有授权码？点击「购买授权码」按钮跳转到购买页。

### 2. 登录 B 站

1. 点击「登录 B 站」按钮
2. 在弹出的窗口中登录你的 B 站账号
3. 登录成功后，点击窗口右上角的「登录完成，提取 Cookie」按钮
4. 看到提示「B 站登录成功！」即可

<!-- TODO: 补充登录窗口截图 -->

> 注意：登录信息会保存在插件中，无需每次重新登录。如果登录过期，同步时会自动提示你重新登录。
> 切换 B 站账号时插件会清空已同步状态，从新账号重新开始同步。

### 3. 同步设置

| 设置项 | 说明 | 默认值 |
|-------|------|--------|
| 根目录名 | 同步内容存储在 Vault 中的目录名 | `BiliBili` |
| 定时自动同步 | 开关。开启后按下方间隔自动同步；遇到限流或登录失效会自动关闭 | 开启 |
| 同步间隔 | 每隔多少分钟自动同步一次（最小 10 分钟） | 30 分钟 |
| 每批同步数量 | 每次同步的视频数量（建议 5-10） | 5 |
| 同步逐字稿 | 是否拉取视频字幕作为正文。无字幕的视频会跳过此项 | 开启 |

<!-- TODO: 补充同步设置截图 -->

### 4. AI 总结（可选）

| 设置项 | 说明 | 示例 |
|-------|------|------|
| 启用 AI 总结 | 开关。开启后会对视频字幕生成 AI 总结，写在笔记顶部「🤖 AI 总结」一节里 | 关闭 |
| API Key | OpenAI 兼容接口的 API Key | `sk-xxxxxx` |
| API Base URL | OpenAI 兼容接口地址，**必须包含 `/v1`** | `https://api.openai.com/v1` |
| 模型 | OpenAI 兼容模型名 | `gpt-4o-mini` |

支持任何 OpenAI 兼容的 API：OpenAI、DeepSeek、Qwen、Gemini、OpenRouter、SiliconFlow 等。

填好后点击「测试连接」，看到「✅ AI 配置可用！」即可。

> 注意：AI 总结依赖逐字稿，所以一定要先开「同步逐字稿」开关。

## 三、开始同步

### 自动同步

启用插件并完成登录后，插件会按你设定的间隔自动同步，无需手动操作。

### 手动同步

- **Ribbon 图标**：点击左侧栏的 B 站图标立即同步
- **命令面板**：`Cmd+P`（macOS）或 `Ctrl+P`（Windows） → 搜索「BiliBili Sync: 立即同步」

### 同步状态

- 同步成功会弹出 Notice 提示：「收藏夹「xxx」：同步了 N 条」
- 设置页可看到「已同步：N 条」+ 状态文本：
  - 「历史数据同步中...」：还在拉取历史收藏夹的过程中
  - 「至少一个收藏夹已全量同步完成，开始增量同步」：已经完成至少一个收藏夹的全量同步，进入增量阶段（只拉新增的收藏）

> 设置页面不会自动刷新，同步完成后需要关闭再重新打开才能看到最新数量；以 Notice 通知为准。

## 四、同步内容说明

### Vault 目录结构

```
BiliBili/
├── Bookmarks/
│   ├── 我的收藏夹/
│   │   ├── 视频标题1-resourceId.md
│   │   └── 视频标题2-resourceId.md
│   └── 学习专区/
│       └── 视频标题3-resourceId.md
└── Media/
    └── {resourceId}/
        └── cover.jpg
```

每个 B 站收藏夹对应一个子目录，里面是该收藏夹下的视频笔记。

### 笔记格式

笔记内容包含：

1. **YAML 属性**（resourceId、bvid、author、url、favorite、publishedAt、favoritedAt、syncedAt）
2. **视频标题**（H1）
3. **封面图**（嵌入 `BiliBili/Media/` 目录下的本地文件）
4. **B 站官方播放器**（`<iframe>` 嵌入，可在 Obsidian 中直接播放）
5. **元信息**（UP 主、收藏夹、原文链接、简介）
6. **🤖 AI 总结**（可选，如果开启了 AI 总结且字幕拉取成功）
7. **章节**（如果视频有章节信息）
8. **逐字稿**（如果开启了「同步逐字稿」且视频有字幕）

## 五、常见问题

### 登录过期

同步时若遇到 cookie 失效，插件会自动关闭定时同步并弹出 Notice 提示。请重新登录后在设置页打开「定时自动同步」。

### 字幕拉不到

- B 站很多视频没有字幕（尤其是早期视频或非 UP 主上传的字幕版本）。
- 部分视频字幕需要登录才能拉，请先确认已登录。
- 控制台会输出 `[bili-subtitle][diag]` 诊断日志，可在 `Cmd+Option+I` 打开开发者工具查看。

### 封面/图片不显示

- 检查 Vault 里的 `BiliBili/Media/{resourceId}/` 目录是否有 `cover.{ext}` 文件。
- 如果文件存在但 Obsidian 显示不了，关一下 Obsidian 重新打开试试。

### 重新同步所有数据

设置页 → 点击「清除缓存并重新同步」按钮（红色）。这只会清除已同步记录，不会删除已有笔记文件，重新同步时会覆盖更新。

### 切换账号

切换登录账号后，插件会自动检测到 DedeUserID 变化，并清空所有已同步状态，从新账号的收藏夹开始重新同步。

### 同步间隔建议

- 最小 10 分钟，建议 30 分钟以上。
- 间隔太短容易触发 B 站频率限制（HTTP 412/461），插件会自动关闭定时同步。

### 插件更新

下载新版 zip → 解压 → 覆盖旧的 `.obsidian/plugins/bili2obsidian/` 目录 → 在 Obsidian 设置 → 第三方插件中点击刷新或重新启用。
